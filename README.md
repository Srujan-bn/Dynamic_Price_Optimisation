# Dynamic Price Optimization System

A machine learning-based pricing model that analyzes historical and 
competitor data to generate data-driven price recommendations.

## Tech Stack
- Python
- Machine Learning (scikit-learn)
- Streamlit

## Features
- Real-time price optimization based on historical data
- Competitor price analysis
- Interactive dashboard for pricing decisions

## How to Run
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Run: `streamlit run app.py`
